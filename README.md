# Sami Fraunholz Website

Next.js + Tailwind Website für Sozialcoaching und ASOG Consulting.

## Lokal starten

```bash
npm install
npm run dev
```

Dann öffnen: http://localhost:3000

## Deploy auf Vercel

1. Dieses Projekt in ein GitHub Repository hochladen.
2. Bei https://vercel.com mit GitHub einloggen.
3. "New Project" auswählen.
4. Repository importieren.
5. Deploy klicken.

## Bilder ersetzen

Lege Bilder in `public/images/` ab und ersetze die Platzhalter in `src/app/page.tsx` später durch Next/Image Komponenten.
